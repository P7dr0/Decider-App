# Decider-App
Decider App é o meu primeiro projeto independente utilizando PHP
A ideia é simples: acabar com aquela indecisão na hora de escolher um filme para assistir
Prático e usual, a aplicação usará PHP, JS, HTML e CSS, além claro de bando de dados SQL

A proposta inicial é usar o projeto como estudo, mas a ideia de subí-lo como site ou apk algum dia não está descartada
